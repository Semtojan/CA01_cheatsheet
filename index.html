HTML cheatsheet

Tables

Tag Name Description
<table> Table element The wrapper element for all HTML tables.
<thead> Table head The set of rows defining the column headers in a table.
<table> Table body The set of rows containing actual table data.
<tr> Table row The table row container.
<td> Table data The table row container.
<th> Table head cell header cells are bold and center aligned
<tfoot> Table foot The set of rows defining the footer in a table.

CSS cheatsheet

Display and positioning

Property Value Description
z-index integer specifies how far back or how far forward an element will appear on a web page when it overlaps other elements
position static, relative, absolute, fixed, sticky options for positioning HTML elements on a web page
display block, inline, inline-block, flex, grid, ... determines the type of render block for an element, e.g. Block elements take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted
float left, right determines how far left or how far right an element should float within its parent element
clear left, right, both, none, ... specifies how an element should behave when it bumps into another element within the same containing element.The clear is usually used in combination with elements having the CSS float property. This determines on which sides floating elements are allowed to float.
